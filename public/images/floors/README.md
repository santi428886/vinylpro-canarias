# Patrones CSS del catálogo

El catálogo de productos **no usa fotografías**. Cada producto muestra un patrón CSS
generado por `VinylFloorPattern`.

## Archivos

| Archivo | Qué editar |
|---------|------------|
| `src/data/floor-patterns.ts` | Colores, estilos y asignación producto → categoría |
| `src/components/ui/VinylFloorPattern.tsx` | Componente visual |

## Categorías

Roble claro/medio/oscuro · Nogal · Gris claro/oscuro · Espiga clara/oscura ·
Hormigón claro/oscuro · Piedra beige/gris · Blanco nórdico · Negro premium

## Fotos reales (futuro)

Cuando tengas fotos de fabricante, añade soporte en `VinylFloorPattern` para
rutas locales bajo `/public/images/floors/`.
